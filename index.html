Java Script ノート

基礎文法

言語 説明
<script></script> htmlファイルの中にjavascriptを書く方法
ソース属性src="〇〇/main.〇〇"でjava scriptを付与できる
"use strict" エラーチェックをするための言語
console.log('') ブラウザのコンソールに値を表示する為の命令
/ /＝’、/n=改行、/t=タブなど
四則演算+α + = 足し算、- = 引き算、* = かけ算、/ = 割り算、% = 整数で割った余り、** = 乗数
const 〇〇 = △△ 右の値(△△)を左の定数(〇〇)に割り当てるという意味(=代入)　再代入は不可
定数を使うとプログラムが変更に強くなる
let 〇〇 = △△ 値の再代入が可能(let 〇〇 = △△の後、〇〇 = ✕✕が可能) 2度使いはNG
varという書き方もあるが古く問題あり
定数/変数の命令規則 その１ 数値から始められない 英数字、＄、＿のみ可
大文字小文字は区別される
予約語は使えない(letにconstを代入などは不可)
定数/変数の命令規則 その２ let price = 500 の後に price = price +100 としたい場合はprice +=100 で表現できる
price *= かけ算、price++ で+1、price-- で-1
定数/変数の命令規則 その２ let i = 0 の後に console.log(i++)++ はあとから実行されるのでconsole.log()に渡されるのはiのみになってしまう(=0) (++i)は1になる
